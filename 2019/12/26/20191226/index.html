64位ubuntu官方系统安装宝塔7.0.3 并成功编译安装nginx+php+mysql环境

2019-12-26
环境搭建-树莓派

终于成功的在64位树莓派环境下安装好了宝塔的lnmp环境，给大家填下坑

直接官方下载ubuntu-18.04.3-preinstalled-server-arm64+raspi3系统

下载地址

https://wiki.ubuntu.com/ARM/RaspberryPi

烧录系统

直接用Win32DiskImager写入（百度一堆不多说，不会的自己百度）

开机后修改软件源

把/etc/apt/sources.list内的文件内容换为

#清华大学源
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu-ports/ bionic main restricted universe multiverse
deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu-ports/ bionic main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu-ports/ bionic-updates main restricted universe multiverse
deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu-ports/ bionic-updates main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu-ports/ bionic-security main restricted universe multiverse
deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu-ports/ bionic-security main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu-ports/ bionic-backports main restricted universe multiverse
deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu-ports/ bionic-backports main restricted universe multiverse

然后

apt-get update && apt-get dist-upgrade

安装宝塔面板

wget -O install.sh http://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh

试了一下 普通的SD卡每次装完基本都在40分钟左右

安装nginx

在面板中先安装nginx 然后等到/www/server/panel/install文件夹内生成nginx.sh文件时 关闭面板安装

然后删除nginx.sh文件

rm nginx.sh

重新下载编译好的nginx.sh文件内容

wget https://www.mzxh.xyz/tools/nginx.sh

然后直接在/www/server/panel/install文件夹下执行

sh nginx.sh install

Nginx安装并启动成功

可能会遇见的报错

./configure:  error: the HTTP image filter module requires the GD library.You can either do not enable  the  module or install the l ibraries
make: *** NO rule to make target ' build', needed by' default'. Stop
make: *** No rule to make target  install'. Stop.

解决办法

sudo apt-get install libgd-dev

然后再次执行安装

安装Mysql

Mysql5.5 测试面板可以直接安装成功

安装php

修改/www/server/panpel/install/lib.sh文件，删除掉最底部的这三条命令

Install_Libmcrypt
Install_Mcrypt
Install_Mhash

删除后手动安装libmcrypt-dev

apt install libmcrypt-dev

下载编译好的php.sh文件

wget https://www.mzxh.xyz/tools/php.sh

然后运行命令安装php7.1（根据你的需求安装php版本）

sudo sh php.sh install 7.1

安装成功
