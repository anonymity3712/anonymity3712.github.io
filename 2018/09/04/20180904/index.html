    第二节：kali的配置及常用软件安装

    2018-09-04
    kali-kali

    虽然在kali中已经预装了超过300种的应用程序，但是有时我们仍需要会安装一些程序来保证平时的基础使用，本篇主要写了kali开机后黑屏处理方法;更新源文件并更新内核头至最新（2018.8.21）;安装Deepin Scrot截图工具;安装wps office;安装搜狗输入法;安装NVIDIA;安装Shadowsocks-qt5;安装Google浏览器;安装 Teamviewer13;安装Virtualbox。。。。。。

    kali开机后黑屏处理方法

    部分双显卡电脑初次开机后可能会出现输入完账号密码后黑屏无法进入系统的问题，这是因为Kali预装的开源显卡驱动nouveau，启动出现了问题，我们只需要禁用nouveau即可，并添加其到启动的黑名单里即可。
    在进入如下引导界面时按下按下e键，进行编辑grub命令。

    在倒数第三行加入了 nouveau.modeset=0 （也就是字符quite的后面）

    然后按下ctrl键+X键，进行启动即可，然后输入登录账户密码，成功进入系统

    更新源文件并更新内核头

    进入系统后进行对系统更新

    编辑:sources.list文件

    leafpad /etc/apt/sources.list

    使用官方的更新源

    deb http://http.kali.org/kali kali-rolling main non-free contrib

    然后输入以下命令

    apt-get clean //清除缓存索引

    apt-get update //更新索引文件

    apt-get upgrade //更新实际的软件包文件

    apt-get dist-upgrade //根据依赖关系更新

    把grub命令添加到黑名单

    为了避免每次开机后都要修改grub命令，我们将nouveau添加到黑名单，防止它启动

    进入/etc/modprobe.d文件夹

    cd /etc/modprobe.d

    创建一个nvidia-graphics-drivers.conf文件

    sudo vi /etc/modprobe.d/nvidia-graphics-drivers.conf

    写入： blacklist nouveau

    保存并退出: wq!

    然后修改/etc/default/grub文件

    sudo vi /etc/default/grub

    末尾写入：rdblacklist=nouveau nouveau.modeset=0

    保存并退出: wq!

    然后再输入 init 6 重启电脑

    重启后可以通过 uname -a 来查看当前版本

    安装Deepin Scrot截图工具

    1.下载Deepin Scrot

    wget http://packages.linuxdeepin.com/deepin/pool/main/d/deepin-scrot/deepin-scrot_2.0-0deepin_all.deb

    下载最新版本的Deepin Scrot软件

    2.更换源文件

    leafpad /etc/apt/sources.list

    注释掉官方源，新增下面中科大源

    deb http://mirrors.ustc.edu.cn/kali kali-rolling main contrib non-free

    3.安装Deepin Scrot

    sudo dpkg -i deepin-scrot_2.0-0deepin_all.deb

    会提示缺少依赖的问题

    用apt --fix-broken install 强制安装依赖关系

    再重新 dpkg -i deepin-scrot_2.0-0deepin_all.deb

    没有报错，安装成功

    名称：截图

    命令: deepin-scrot

    设置快捷键

    点击蓝字添加；就完成了

    安装wps office

    点击此处，下载安装包，32位电脑下载32位，64位电脑下载64位，下载以.deb结尾的那一个。

    在终端输入安装命令。

    dpkg -i wps-office_10.1.0.6634_amd64.deb

    就成功了，这个时候运行wps会出现报错问题，是因为linux下缺失wps运行时需要的字体

    下载文字 https://pan.baidu.com/s/1eSoLzjk 下载后，把 wps-office提取到 /usr/share/fonts文件夹中，再次运行就不会报错了。

    安装搜狗输入法

    更换源

    vi /etc/apt/sources.list
    deb http://cn.archive.ubuntu.com/ubuntu/ trusty main restricted universe multiverse
deb http://cn.archive.ubuntu.com/ubuntu/ trusty-security main restricted universe multiverse
deb http://cn.archive.ubuntu.com/ubuntu/ trusty-updates main restricted universe multiverse
deb http://cn.archive.ubuntu.com/ubuntu/ trusty-backports main restricted universe multiverse

    添加公钥

    apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 3B4FE6ACC0B21F32

    然后在更新

    apt-get update

    安装fcitx小企鹅以及相关依赖包的命令如下

    apt-get install fcitx-libs fcitx-libs-qt fcitx-bin fcitx-data fcitx-modules fcitx-module-dbus fcitx-module-kimpanel fcitx-module-lua fcitx-module-x11 fcitx fcitx-tools fcitx-ui-classic fcitx-config-gtk2 libopencc1

    然后再去https://pinyin.sogou.com/linux/ 这个网站下载deb的安装包

    下载后，使用dpkg -i sogoupinyin_2.2.0.0108_amd64.deb命令安装就行了，

    安装好后重新启动init 6，会看到一只小企鹅

    输入法安装成功

    安装NVIDIA

    更换kali的源，用中科大源。

    deb http://mirrors.ustc.edu.cn/kali kali-rolling main contrib non-free

    下载安装N卡驱动

    apt install nvidia-driver nvidia-xconfig

    查看电脑中 N 卡的总线 ID

    nvidia-xconfig --query-gpu-info | grep 'BusID : ' | cut -d ' ' -f6

    记录如下查询结果 PCI:1:0:0

    在/etc/X11中创建一个xorg.conf文件

    vi /etc/X11/xorg.conf

    填入如下信息：

    Section "ServerLayout"
 Identifier "layout"
 Screen 0 "nvidia"
 Inactive "intel"
EndSection

Section "Device"
 Identifier "nvidia"
 Driver "nvidia"
 BusID "PCI:1:0:0"
EndSection 

Section "Screen"
 Identifier "nvidia"
 Device "nvidia"
 Option "AllowEmptyInitialConfiguration"
EndSection

Section "Device"
 Identifier "intel"
 Driver "modesetting"
EndSection

Section "Screen"
 Identifier "intel"
 Device "intel"
EndSection

    然后创建/usr/share/gdm/greeter/autostart/optimus.desktop 文件

    vi /usr/share/gdm/greeter/autostart/optimus.desktop

    填入如下信息：

    [Desktop Entry]
Type=Application
Name=Optimus
Exec=sh -c "xrandr --setprovideroutputsource modesetting NVIDIA-0; xrandr --auto"
NoDisplay=true
X-GNOME-Autostart-Phase=DisplayServer

    创建 /etc/xdg/autostart/optimus.desktop 文件

    vi /etc/xdg/autostart/optimus.desktop

    填入如下信息：

    [Desktop Entry]
Type=Application
Name=Optimus
Exec=sh -c "xrandr --setprovideroutputsource modesetting NVIDIA-0; xrandr --auto"
NoDisplay=true
X-GNOME-Autostart-Phase=DisplayServer

    重启reboot

    可以看到已经更新成N卡

    安装Shadowsocks-qt5

    1.添加源，其他源可以注释，添加这一条源，注意：官方源不能注释，不然会提示缺少依赖

    vi /etc/apt/sources.list
    deb http://ppa.launchpad.net/hzwhuang/ss-qt5/ubuntu devel main
deb http://http.kali.org/kali kali-rolling main non-free contrib

    2.使用apt-get update更新一下源

    apt-get update


    会出现密匙过期的问题，这时候我们添加一个密匙就好了

    apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 6DA746A05F00FA99
///6DA746A05F00FA99【这个是对应上面报错的NO_PUBKEY后面的字符】或许你的会不一样，记得变通下

    再次执行更新命令

    apt-get update

    3.下载会需要的依赖包 https://pan.lanzou.com/i0t76jg

    下载后解压安装deb包

    dpkg -i libqtshadowsocks_1.10.0-1_amd64.deb libbotan-1.10-0_1.10.8-2+deb8u1_amd64.deb libssl1.0.0_1.0.1t-1+deb8u7_amd64.deb


    4.安装shadowsocks-qt5

    apt-get install shadowsocks-qt5

    5.修改文件vi /etc/proxychains.conf

    vi /etc/proxychains.conf

    打开后，修改下规则，去掉dynamic_chain前面的#号，把socks4改成socks5即可，端口选择你喜欢的就好，建议选8000之后，待会配置shadowsock-qt5需要对应的

    dynamic_chain
socks5 127.0.0.1 8089

    记得保存退出:wq

    6.安装完成，可以添加节点测试

    点击连接-添加-手动

    服务器地址和端口填写为你的SS服务器的连接IP地址和端口

    本地端口改成和你在proxychains中的socks5对应的端口即可

    填写完毕后，选择节点名字【蓝色标示】然后点击那个黄色的五角星，如果出现延迟【我的是48ms】


    就说明你的SS梯子配置没有问题了，不正常的话要去检查一下输入的ss账号信息是否有误。

    7.开启科学上网模式，使用google搜索你想要的答案

    安装 proxyresolv

    cp /usr/lib/proxychains3/proxyresolv /usr/bin/

    然后再测试下google

    proxyresolv www.google.com

    返回ok，说明配置正确，可以愉快的跨栏了。

    4).使用Proxychains命令启动你想启动应用程序

    打开一个终端，输入

    proxychains firefox
##使用这个proxychains命令的记得先关掉原来开的火狐浏览器
##其他nmap和sqlmap也是如此，这样才会代理成功

    就会打开Kali自带的火狐浏览器了，然后浏览器地址栏输入：https://www.google.com.hk 即可（此时一定不要关闭终端窗口，关掉后firefox也会停止运行）

    使用ProxyChains代理其他程序和启动Firefox同理

    proxychains msfconsole //启动msf

    proxychains sqlmap //通过代理启动sqlmap

    还有很多用法，都可以用proxychains这个命令+Shadowsocks-qr5实现代理哦

    强制安装依赖关系

    再次安装成功

    这时候如果是root账号，运行软件可能会出现双击没反应的情况，进入/usr/share/applications 找到浏览器图标 在命令后加 --no-sandbox

    再次双击图标就可以正常运行了

    安装 Teamviewer13

    点击此处下载Teamviewer13安装包。

    用dpkg命令安装

    dpkg -i teamviewer_13.1.8286_amd64.deb

    提示报错，缺少依赖环境。用 apt --fix-broken install 命令强制安装依赖包

    再次运行 安装成功

    安装Virtualbox

    先执行下面命令

    leafpad /etc/apt/sources.list

    将软件源替换为官方源

    deb http://http.kali.org/kali kali-rolling main non-free contrib

    更新一下更新索引文件

     apt-get update

    然后安装virtualbox

     apt-get install virtualbox -y

    若最后出现DKMS: install completed.表示安装完成

    在终端输入 virtualbox 即可开启虚拟机。（在软件列表中，点击virtualbox的图标也可以运行）
