    vulfocus0.2.1新版本本地化部署

    2020-05-12
    环境搭建-Docker

    Vulfocus 是一个漏洞集成平台，将漏洞环境 docker 镜像，放入即可使用，开箱即用。

    一.配置环境：

    ubuntu19.10 service 镜像下载地址：传送门

    1.修改源为阿里源

    进入/etc/apt目录下，会有一个sources.list文件，先把他备份一份sudo cp sources.list sources.list.old 然后打开sources.list

    sudo vim sources.list

    替换为以下内容

    deb http://mirrors.aliyun.com/ubuntu/ eoan main restricted universe multiverse
deb-src http://mirrors.aliyun.com/ubuntu/ eoan main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ eoan-security main restricted universe multiverse
deb-src http://mirrors.aliyun.com/ubuntu/ eoan-security main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ eoan-updates main restricted universe multiverse
deb-src http://mirrors.aliyun.com/ubuntu/ eoan-updates main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ eoan-backports main restricted universe multiverse
deb-src http://mirrors.aliyun.com/ubuntu/ eoan-backports main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ eoan-proposed main restricted universe multiverse
deb-src http://mirrors.aliyun.com/ubuntu/ eoan-proposed main restricted universe multiverse

    执行命令sudo apt-get update和sudo apt-get upgrade

    2.安装所需环境

    安装docker

    官方安装步骤

    https://docs.docker.com/engine/install/ubuntu/

    从Install using the repository步开始安装即可

    安装pip3
    apt install python3-pip
    安装nginx
    apt install nginx
    安装npm
    apt install npm
    安装redis-server
    apt install redis-server

    3.修改docker配置

    修改/usr/lib/systemd/system/docker.service

    在[Service]中注释掉原有的ExecStart=并新加一条

    ExecStart=/usr/bin/dockerd -H tcp://127.0.0.1:2375 -H unix:///var/run/docker.sock

    4.修改nginx配置

    配置上传文件大小，修改/etc/nginx/nginx.conf文件，在 http 中加入：

    client_max_body_size 4096M;

    其中 4096M（4GB） 为上传文件最大限制，可根据实际进行修改，最小配置为 200M 。

    修改 nginx 配置目录 sites-available 中 default 文件（/etc/nginx/sites-available/default） ，server 节点添加以下代码：

    location /api/ {
        proxy_pass http://127.0.0.1:8000/;
}

    二.部署后端项目 vulfocus-api

    项目地址：传送门

    直接拖拽到服务器上

    git clone https://github.com/fofapro/vulfocus.git

    进入vulfocus-api文件夹

    安装依赖
    pip3 install -r requirements.txt

    修改配置

    修改vulfocus-api/vulfocus/settings.py文件

    配置 VUL_IP，修改为 Docker 服务器的 IP。

    启动 Celery

    在 vulfocus-api 中启动 Celery：

    celery multi start worker -A vulfocus -l info --logfile=celery.log

    然后执行ps -ef |grep celery

    出现红框中的内容便代表celery正在运行中

    部署前端项目 Vulfocus Frontend

    vulfocus 前端项目，通过 Element-ui + VUE 构建。

    部署

    项目构建，进入vulfocus-frontend文件夹

    安装依赖：
    npm install

    构建项目：
    npm run build:prod

    把 dist 文件夹下的内容复制到 nginx 静态目录下 /var/www/html

    重启nginx服务
    sudo /etc/init.d/nginx restart

    启动 API 后端：

    进入vulfocus-api文件夹

    python3 manage.py runserver 0.0.0.0:8000

    访问靶机IP的80端口

    默认账号密码均为admin

    初次安装后没有漏洞镜像的需要从 https://hub.docker.com/ 网站拉取镜像

    具体操作见官方手册：传送门
